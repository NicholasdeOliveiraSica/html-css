Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (bold=negrito)

Já nesta frase temos um termo em destaque, usando a tag STRONG (forte)

Italico / ênfase

Nesta frase temos um termo em Italico, usando a tag I

Já nesta frase temos um termo com ênfase, usando a tag Em

Conclusão

Visualmente é a mesma alteração, porém o HTML5 lida com conteúdo, portanto quaisquer alterações visual devem ser tratados semanticamente, e não sobre a forma do texto em si, "o texto não está em NEGRITO", o texto esta em DESTAQUE, "O texto não está em Italico", o texto tem ênfase

Marcado

Para deixar o texto MARCADO, usa-se <mark>, para alterar a cor, deve-se criar uma folha de estilo, style=background-color: cor, tipo apaixo estará azul

MARCADO AZUL

Grande / Pequeno

Como deixar a palavra grande não é algo semantico, não há tags para isso, porém para palavras pequenas existe a tag <small>
